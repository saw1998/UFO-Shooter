# UFO-Shooter
A c++ game from scratch

# Installation
* install sfml on your machine
  * ref: https://www.sfml-dev.org/tutorials/2.6/
* clone the repo
  * $ git clone https://github.com/saw1998/UFO-Shooter.git
* navigate to cloned directory
* create a build directory
  * $ mkdir build
* build using cmake
  * $ cmake ..
  * make
* Execute
  * $ ./UFO-Shooter
